Buttons & Clicks


Buttons & Button IDs:
In my webpage, I used buttons to show the countries and button IDs to create variables to get the colorchange and display. I had to use
buttons to click and show the countries. In the webpage, I created a button/button id in HTML and then created a variable in Javascript
with the ID's. Using the variables, I was able to do a style display change where when you click on a specific continent, other countries
disappear so that the countries wouldn't get jumbled. I also had it so that the countries would appear when a specific continent was
clicked. The tag I used for the button id's and buttons was <button id="asia">Asia</button> for the continent
of asia. I also created one for the other 6 continents. Then I made many other buttons for the countries within those continents. I used
a list tag to order the continents and had the countries below the heading of Continents/Places I've Been. Using getElementById I was
able to connect HTML with my javascript and by using eventlisteners I was able to have my display of countries appear and disappear when
the buttons are clicked.


Color change:


Onclicks:
As I was making my webpage, I decided I wanted my buttons to change colors when clicked so I decided to use an onclick. I connected my
onclick with my button id for the countries so that when clicked they would turn red. I used the code
<button id="japan" onclick="changing(this, 'red')">Japan</button>
to get the color change as red. I could have used blue or green but I found that red stands out. For the onclick to work, I had to use
a function in my HTML that is a style color change function. The code was:
<script>
function changing(elmnt,clr) {
elmnt.style.color= clr;
}
</script>

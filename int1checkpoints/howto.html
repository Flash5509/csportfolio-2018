List Tags:


Why:
In my about me page, I have an ordered list as well as an unordered list.
This helps organize what I want to be stated where an ordered list has numbers
and an unordered list has bullet points.
How:
Using the tag <ol> and <ul>, the <ol> tag is used to create an ordered list.
The <ul> tag is used to create an unordered list. The <li></li> is what's
used to actually put what you want inside the list.
For example:
Putting the codes:
<ol>
<font size=4px>
<li>How to change background color & different colors </li>
<li>Changing the font. </li>
<li>How to create lists and tables </li>
<li>Adding images & changing the size of the images </li>
<li>Adding border and spacing between images </li>
<li>Changing borders and adding background color in words </li>
</ol>

This will create an ordered list:

  1. How to change background color & different colors
  2. Changing the font.
  3. Creating list tags.
  4. Adding images & changing the size of the images
  5. Adding border and spacing between images
  6. Changing borders


Body Tag:
(the things inside:))


Why:
The body tag is what defines the document's body and the content inside. From
having spacing to having paragraphs as well as headings.
How:
Using the tag <body>, <br>, <p>, and <h1>, this makes it
so that you have a place to put what you want to write, have spacing, write a
paragraph with a small font, and have a heading in different sizes depending on
which heading you want. The range of headings go from <h1> to <h6>.
For example:
Putting the code:
<body>
<bold>Facts</bold>
</h1>
<p>
I am a sophmore at Summit Rainier High School. I love reading and music.
<br>
</p>
</body>

This will create a body:


Facts

I am a sophmore at Summit Rainier High School. I love reading and music.


Hyperlinks (tag):


Why:
Hyperlinks are used to bring you to a different page quickly in a click without
having to type in a new url.
How:
Using the tag <a>, this makes a hyperlink which depending on how you use it,
it can bring you to a different page or just bring you to a different part of
the webpage.
For example:
Putting the code:
<ul id="navbar">
<li><a href=index.html"> About Me </a></li>
<li><a href=favoritethings.html"> Favorite </a></li>
</ul>
